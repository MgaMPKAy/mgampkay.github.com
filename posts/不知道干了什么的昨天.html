一点WordPress

替换WordPress文章中的URL，简单的文本替换都可以这么干：

UPDATE wp_posts SET post_content = REPLACE (post_content, 'http://from.com', 'http://to.net');

重设管理员密码，直接改，用MD5加密：

UPDATE  wp_users SET  user_pass = MD5('password') WHERE  wp_users.ID = 1 LIMIT 1 ;
